程序实例

感知机原始形式

这个文件是《统计学习方法》(李航)一书中对应的例子.

这里用到numpy, 以支持向量运算.

设置数据集

训练数据集, $x_1=(3,3)^T$, $x_2=(4,3)^T$为正例, $x_3=(1,1)^T$为负例.

设置初始运行参数

设置初始运行参数$w_0=0$, $b_0=0$, 学习率$\eta$用learning_rate表示.

判断表达式

判断表达式的计算: $y_i(w \cdot x_i +b)$:

  • item[0]表示$x_i$
  • item[1]表示$y_i$

$w$和$b$的更新

当$y_i(w \cdot x_i +b) \leq 0$时, 更新$w$和$b$:

  • $w \leftarrow w + \eta y_i x_i$
  • $b \leftarrow b + \eta y_i$

感知机原始形式计算

w =  [3 3]   b =  1
w =  [2 2]   b =  0
w =  [1 1]   b =  -1
w =  [0 0]   b =  -2
w =  [3 3]   b =  -1
w =  [2 2]   b =  -2
w =  [1 1]   b =  -3
RESULT： w =  [1 1]   b =  -3

直线方程表示为: $w \cdot (x, y)^T + b = 0$, 即 $w_0 x + w_1 y +b =0$.

所以,

$$y = \frac{-(w_0x+b)}{w_1}$$
